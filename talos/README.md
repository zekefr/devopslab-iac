# Talos Folder

This folder stores Talos local inputs and generated artifacts.

- `cluster.generated.env`: generated cluster config from Terraform output (ignored by Git).
- `cluster.local.env.example`: optional local override sample.
- `cluster.local.env`: optional local override file (ignored by Git).
- `generated/`: generated machine configs (ignored by Git).

See `docs/30-talos.md` for the full workflow and commands.
